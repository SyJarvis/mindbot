## ⚙️ 实时配置系统

配置热更新，授权实时生效，无需重启服务。

### 核心组件

| 组件 | 职责 |
|------|------|
| `ConfigBus` | 内存事件总线，配置读写零延迟，订阅通知机制 |
| `AuthManager` | 实时授权管理，工具授权立即生效，支持临时授权自动过期 |
| `ConfigPersistence` | 周期性持久化（默认 60 秒），崩溃恢复不丢数据 |
| `ConfigSync` | 多实例同步（可选），版本号心跳 + 增量拉取 |

### 使用示例

```python
from mindbot.config import ConfigBus, AgentConfigIntegration

# 创建配置集成
integration = AgentConfigIntegration()
await integration.initialize()

# 设置配置（立即生效）
await integration.set("global", "temperature", 0.9)

# 授权工具（1 小时后自动过期）
await integration.grant_auth("user_123", "delete_file", expires_in=3600)

# 检查授权（实时查询）
allowed, reason = await integration.check_auth("user_123", "delete_file")
```

### CLI 支持

```bash
mindbot shell

# Shell 内实时配置
/config get global temperature
/config set global temperature 0.9
/config auth grant user_123 delete_file --expires 3600
/config auth check user_123 delete_file
```

## 🔌 ACP 协议支持

Agent Client Protocol 通道，支持 Claude Code、Codex 等外部 Agent。

### 功能特性

- **ACPChannel** — 连接 ACP 兼容的 Agent（Claude Code、Codex）
- **智能路由** — 按通道/会话 ID 匹配规则，将消息路由到指定 Agent
- **无缝集成** — 飞书/Telegram 消息可自动转发到外部 Agent 处理

### 配置示例

```json
{
  "channels": {
    "acp": {
      "enabled": true,
      "default_agent": "claude-code",
      "agents": {
        "claude-code": {
          "name": "Claude Code",
          "command": "npx",
          "args": ["-y", "@anthropic/claude-code-acp"],
          "cwd": "/path/to/project"
        }
      },
      "routing": {
        "feishu:*": "claude-code"
      }
    }
  }
}
```

**效果**：飞书所有会话的消息自动路由到 Claude Code 处理。

## 📊 Agent Benchmark

量化评估 Agent 性能，支持工具调用和真实执行测试。

### ToolCall-15

验证 Agent 工具使用能力，包含 15 个工具调用场景。

```bash
# 启动适配器
mindbot toolcall15-adapter --host 127.0.0.1 --port 11435 --model local-ollama/qwen3

# 运行 benchmark
cd benchmark/ToolCall-15
npm install && npm run dev
```

### Real-Tools

验证真实文件/Shell/HTTP 工具执行效果。

```bash
python benchmark/real-tools/runner.py \
  --config-path ~/.mindbot/settings.json \
  --model local-ollama/qwen3
```

### 输出报告

- JSON 格式报告（`benchmark/real-tools/reports/latest.json`）
- 场景通过率、工具调用统计、错误分析
