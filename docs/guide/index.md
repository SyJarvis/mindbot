# 使用手册

欢迎使用 MindBot 使用手册。MindBot 是一个基于 Python + asyncio 的模块化 AI Agent 框架，支持多 Provider、动态路由、流式响应和工具确认机制。

本手册将帮助你完成从安装到进阶使用的全过程。
